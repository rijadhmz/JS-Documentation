Uvod

JavaScript je jednostavan, interpretiran programski jezik namijenjen ponajprije razvoju interaktivnih HTML-stranica.

Jezgra JavaScripta uključena je u većinu današnjih preglednika (Internet Explorer, Google Chrome, Mozilla Firefox, Opera, Safari i drugi).

Sta trebate znati

Vodic podrazumijeva da znate sljedece:

  1. Generalno razumijevanje interneta i WWW (World Wide Web),
  2. Dobro razumijevanje HTMla (Hyper Text Markup Language)
  3. Neko iskustvo s programiranjem. Ako ste novi u svijetu programiranja, probajte jedan od tutorijala na nasem sajtu.
JavaScript i Java

Java i Javascript mogu zvučati slično zbog reči „java“, ali ovo dvoje osim toga, nemaju sličnosti. Iako se i Java i Javascript koriste za razvoj web i mobilnih aplikacija, postoje zaista važne razlike između njih.

  • Javu je kreirao 1991. godine James Gosling i od tada je postao jedan od najpopularnijih programskih jezika na svetu. Posle njegovog stvaranja, grupi inženjera Sun Microsistems-a trebalo je 4 godine da razviju Javu. Pored web i mobilnih aplikacija, Java se takođe koristi u ugrađenim sistemima, desktop aplikacijama i u naučnim aplikacijama. 3 milijarde uređaja širom sveta, uključujući računare, štampače, pametne telefone, bankomate, kućne bezbednosne sisteme, koriste programski jezik Java.
  • JavaScript, koji je stvoren 1995. kao skriptni jezik za web, bio je poznat pod nazivom Mocha. Brendan Eich-u je trebalo 10 dana da ga razvije. Nakon nekog vremena njegovo ime je promenjeno u LiveScript, a zatim u JavaScript, uglavnom kao marketinška taktika kako bi ga povezivali sa već popularnim programskim jezikom Java. Osim imena, Java i JavaScript nisu ni na koji način povezani.
Hello World

Kako bi zapoceli pisanjem Javascripta, otvorite Scratchpad i napisite vas prvi "Hello world" Javascript kod:

function greetMe(yourName) { alert("Hello " + yourName); } greetMe("World");

Selektujte kod u padu i pretisnite ctrl+R i pratite vas browser!.

Varijable

Nazivi varijabli i funkcija trebaju zadovoljavati ovo pravilo: prvi znak u nazivu treba biti slovo, podcrta ( _ ) ili dolarski znak ($). Znakovi koji slijede mogu biti slova, brojevi, podcrta ili dolarski znak.

Nazivi ne smiju biti isti kao ključne riječi. Neke od rijeci koje treba izbjegavati kod imenovanja varijabli su: break, else, continue, delete, void i slicno.

Deklaracija varijabli

Varijablu mozete deklarisati na 3 razlicita nacina:
Sa kljucnom rijeci var:

var x = 42.

Ova sintaksa se moze koristiti za lokalne i globalne varijable. Potrebno je samo dodijeliti vrijednost:

x = 42.

Ovo uvijek deklarira globalnu varijablu. Generise striktno JavaScript upozorenje. Ne bi trebali ovo koristiti.
S kljucnom rijeci "let":

let y = 13.

Ovu sintaksu mozete koristiti za deklarisanje block scope local varijable.
